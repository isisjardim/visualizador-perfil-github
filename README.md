# 🔍 Visualizador de Perfil do GitHub

Aplicação web que permite buscar usuários do GitHub e visualizar informações do perfil de forma simples e intuitiva.

## 🚀 Demo

👉 https://isisjardim.github.io/visualizador-perfil-github/

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar consumo de API, manipulação do DOM e organização de código em JavaScript.

A aplicação permite que o usuário digite um username do GitHub e visualize dados reais do perfil.

---

## ⚙️ Funcionalidades

- 🔎 Buscar usuário do GitHub pelo username
- 👤 Exibir avatar, nome e bio
- 👥 Mostrar número de seguidores e seguindo
- 📦 Listar repositórios públicos
- ⚠️ Tratamento de erro (usuário não encontrado)
- ⏳ Estado de carregamento (loading)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- API pública do GitHub

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Consumo de API com `fetch`
- Uso de `async/await`
- Manipulação do DOM
- Tratamento de erros com `try/catch`
- Organização de lógica JavaScript
- Boas práticas de código
